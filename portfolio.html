Bracket Master

Descripción del proyecto

Bracket Master es una plataforma web informativa creada para la comunidad de Neo Smash. ContaSmart

Descripción del proyecto

ContaSmart es una plataforma educativa diseñada para facilitar el aprendizaje del cálculo de nóminas, contabilidad básica y procesos administrativos mediante ejercicios interactivos. Su objetivo es ofrecer una herramienta clara, accesible y práctica para estudiantes que inician en el mundo contable.

La plataforma permite trabajar con ejemplos reales, simular nóminas paso a paso y reforzar conceptos clave, todo dentro de un entorno intuitivo y responsivo desarrollado especialmente para uso académico.

Tecnologías utilizadas
  • Laravel 10 (Arquitectura MVC)
  • SQLite (Base de datos educativa)
  • Blade Templates
  • Bootstrap 5
  • HTML, CSS y JavaScript

Biblioteca Inteligente — Proyecto Arduino

Descripción del proyecto

La Biblioteca Inteligente es un sistema automatizado desarrollado con Arduino que activa la iluminación de forma automática al detectar baja luz ambiental. Además, registra el tiempo total que las luces permanecen encendidas, permitiendo evaluar consumo energético y hábitos de uso.

Este proyecto combina sensores, electrónica básica y programación para ofrecer una solución real aplicable a aulas, bibliotecas, espacios de estudio y entornos donde se necesita iluminación eficiente y automatizada.

Características principales
  • Encendido automático mediante sensor LDR.
  • Registro de tiempo en funcionamiento utilizando temporizadores.
  • Control de iluminación eficiente y confiable.
  • Uso de Arduino para procesamiento y control.
  • Diseño modular, escalable y fácil de mantener.
Tecnologías utilizadas
  • Arduino UNO
  • Sensor LDR para detección de luz
  • Relay para control de lámparas
  • Circuitería eléctrica básica
  • Programación en C++ (Arduino IDE)
