Biblioteca Inteligente

Arduino · IoT

Sistema automatizado con Arduino que activa la iluminación al detectar baja luz ambiental y registra el tiempo de operación. Combina sensores, electrónica y programación para una solución aplicable en aulas y bibliotecas.

Características
  • Encendido automático con sensor LDR
  • Registro de tiempo en funcionamiento
  • Diseño modular y escalable
Tecnologías
  • Arduino UNO
  • Sensor LDR
  • Relay para control de lámparas
  • C++ (Arduino IDE)
