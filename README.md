# Portafolio Personal — Angel David Hinojos Vega

Portafolio web personal desarrollado con HTML, CSS y JavaScript. Publicado en GitHub Pages.

## Secciones

- **Inicio** — Presentación con texto animado y acceso rápido a proyectos y CV.
- **Sobre mí** — Perfil profesional, habilidades y línea de tiempo.
- **Proyectos** — Galería filtrable con modales de detalle para cada proyecto.
- **Curriculum** — Educación, experiencia, habilidades, proyectos y certificaciones.
- **Formación Técnica y Certificaciones** — Insignias digitales verificables (Credly).
- **Contacto** — Formulario de contacto y datos de ubicación.

## Tecnologías

- HTML5, CSS3, JavaScript
- Bootstrap 5
- AOS (animaciones al hacer scroll)
- Isotope (filtrado de portafolio)
- Swiper / GLightbox
- Typed.js
